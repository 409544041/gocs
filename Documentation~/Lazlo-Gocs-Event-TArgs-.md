## Event&lt;TArgs&gt; `type`
Represents an event with arguments.  
Events support assigning handlers and invoking from anywhere.
### Remarks
Use an arguments struct or class if you need to pass multiple arguments.
### Example
Use an arguments struct or class if you need to pass multiple arguments.
